Bus Marker Design Options

Which marker design communicates bus + occupancy best?

Option A — Occupancy Ring

Colored ring around the marker shows fill level. Green=Low, Amber=Medium, Red=Full. Bus icon + plate removed.

Option B — Fill-Level Circle

Circular progress indicator (like a battery) showing occupancy % inside. Centered bus icon stays minimal.

Option C — Badge Bubble

Full plate + route color header. Small colored bubble in top-right shows passenger count. Familiar bus silhouette.
